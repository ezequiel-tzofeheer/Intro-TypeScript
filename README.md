This repository represents the beginning of a structured journey into **typed JavaScript development** and modern software engineering practices.

---

## 📌 What is this?

**Intro TypeScript** is a basic educational project designed to demonstrate the fundamental concepts of TypeScript.

It contains examples and exercises covering:

- TypeScript introduction
- Basic types
- Functions
- Classes
- Interfaces
- Type safety
- Static typing concepts

The focus is on understanding how TypeScript improves code reliability, maintainability and scalability.

## ⚙️ Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/ezequiel-tzofeheer/intro-typescript
```

### 2️ Access the project directory

```bash
cd intro-typescript
```

### 3️ Install dependencies

```bash
npm install
```

### 4️ Compile TypeScript

```bash
npx tsc
```

### ▶️ Run the Project

After compilation:

```bash
node dist/index.js
```
